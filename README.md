# BORODACH — сайт барбершопа (Тольятти)

Премиальный лендинг с онлайн-записью для мужского барбершопа BORODACH.
Фронтенд полностью готов к размещению. Бэкенд подготовлен архитектурно
(контракт API описан ниже), но не реализован — это следующий шаг.

## Стек

- **HTML5 / CSS3 / JavaScript (ES6, без сборщиков)**
- **TailwindCSS** — подключён через CDN, используется точечно для утилит;
  основная вёрстка и уникальный дизайн — в собственных CSS-файлах.
- Шрифты: **Fraunces** (заголовки), **Inter** (текст), **IBM Plex Mono** (цены, лейблы, даты).

```
borodach/

│   ├── booking.js           — состояние и шаги виджета записи + панель цен услуг

│   └── api.js                — обёртка над POST /api/booking

```

## Дизайн-концепция

- **Палитра**: Ink `#0D0C0B`, Charcoal `#17140F` / `#1F1B15`, Bone `#F3EEE3`,
  Brass `#B08D57` / `#D4B27F`, тёмно-зелёный акцент `#2F4F3D`.
- **Контраст текста** соответствует ТЗ: на светлых карточках `#111111` /
  `#555555`, на тёмных блоках `#ffffff` / `rgba(255,255,255,0.8)`.
- **Фирменный элемент — "Cut Line"**: тонкая латунная линия, которая
  "прорезает" экран как штрих опасной бритвы — используется как индикатор
  прогресса скролла, подчёркивание в hero-заголовке, hover-подчёркивания
  ссылок и прогресс шагов записи. Это единственный акцентный жест дизайна —
  всё остальное сдержанно.
- Много воздуха, крупная типографика, аккуратные hover-эффекты на карточках
  мастеров/услуг/галереи, мягкий glassmorphism в шапке при скролле.

## Онлайн-запись

Виджет записи — 5 шагов: мастер → услуги → дата → время → контакты.
Логика полностью на клиенте (`booking.js` + `calendar.js`), без оплаты,
предоплаты, QR или СБП — сайт только принимает заявку.

Отдельно на секции «Услуги» есть быстрый калькулятор стоимости — сумма и
время пересчитываются на лету при выборе чекбоксов и синхронизируются с
виджетом записи при переходе к нему.

## API-контракт (для бэкенда)

Фронтенд отправляет заявку на:

```
POST /api/booking
Content-Type: application/json

{
  "name": "Иван Иванов",
  "phone": "+7 (900) 000-00-00",
  "master": "Алекс",
  "services": ["Мужская стрижка", "Оформление бороды"],
  "date": "2026-07-20",
  "time": "14:20",
  "total": 2500
}
```

Ожидаемый ответ:

```json
{ "ok": true, "id": "booking_123" }
```

Бэкенд должен:
1. Валидировать и сохранить заявку.
2. Отправить уведомление в Telegram (токен бота и chat_id хранятся **только
   на сервере**, никогда не в клиентском JS).
3. Вернуть `{ ok: true, id }` или `{ ok: false, error }`.

Пока `/api/booking` не задеплоен, `js/api.js` автоматически подставляет
локальную симуляцию ответа (см. `simulateLatency`), чтобы весь UX-флоу можно
было демонстрировать без бэкенда. Перед продакшн-запуском удалите этот
fallback в `api.js`.

## Изображения

В текущей версии используются реальные фотографии со стока Unsplash
(портреты барберов, интерьер, работы) — подключены напрямую по URL для
быстрого предпросмотра. Перед продакшеном рекомендуется:
- заменить на собственные фотографии салона и мастеров;
- положить их в `images/` и обновить пути в `index.html`;
- оптимизировать (WebP, компрессия) для скорости загрузки.

- `favicon` (инлайн SVG)
- JSON-LD разметка `HairSalon` (LocalBusiness) с адресом, часами работы и

## Адаптивность

Проверено без горизонтального скролла на брейкпоинтах:
- Desktop (1280px+)
- Ноутбук / планшет-ландшафт (≤1100px)
- Планшет-портрет (≤900px)
- Телефон (≤780px, ≤400px)

На мобильных устройствах появляется плавающая кнопка «Записаться» внизу экрана.

## Доступность и производительность

- Клавиатурный фокус не подавляется нигде в кастомных стилях.
- `prefers-reduced-motion: reduce` отключает все декоративные анимации.
- Изображения используют `alt`-тексты на русском.
- Карта — `iframe` Яндекс.Карт с `loading="lazy"`.

## Как запустить

Статический сайт, бэкенд не требуется для просмотра:

```bash
cd borodach
python3 -m http.server 8000
# открыть http://localhost:8000
```

Для продакшена — разместить папку на любом статическом хостинге
(Vercel, Netlify, Nginx) и добавить бэкенд-обработчик `/api/booking`.
